このリマインダーアプリをdiscordのbot化する

〇目的 : ARKに絞る

〇概要 : Discordにbotをメンションし、コマンドを入力すると、botと対話形式でリマインダーをセットできる
        時間が来た場合、discordチャットの登録者にメンション付きで内容をリマインドする

〇リマインダーの出力内容
    @登録者のdiscord
    〇〇〇〇(リマインドの種類)の時間です！
    [種類] : 〇〇〇〇(テイム、交配...など)
    [内容] : ××××××××××××××××××××××××××××××(恐竜の名前やその他メモ等)

〇botとのやり取り
    [通常]
    １．Bot起動
    ２．「@bot名[半角スペース]set」を入力
    ３．bot「何時間後にリマインダーを設定しますか？」
        bot「例 : 3時間5分後の場合　→　03:05」
    ４．bot「リマインドの種類を設定してください（数字で入力）」
        bot「1.テイム」
        bot「2.孵化・出産」
        bot「3.交配タイマー」
        bot「4.その他」
        →半角数字を入力
    ５．bot「リマインダーの内容を設定してください」
        →文字列入力
    ６．bot「リマインダーの設定が完了しました。」
        bot「bot名では以下のコマンドが使用できます」
        bot「show ... 設定したリマインダーの一覧を表示」
        bot「del ... 設定したリマインダーを削除(番号で指定)」
        bot「upd ... 設定したリマインダーの時間・内容を変更(番号で指定)」
    ７．時刻が来るとbotからメンション付きでリマインドされる
    ８．リマインダーが残っている状態でbotを終了すると、jsonファイル等に残留したリマインダーの内容を全て書きだす
        ※この時、設定時刻から時間が進んだ分計算し、残り時間をjsonに書き出すようにする

    [jsonに内容がある場合]
    １．Bot起動
    ２．bot「前回実行されなかったリマインダーがあります。」
        bot「再度設定したい場合、お手数ですが下記一覧を参照し、もう一度設定しなおしてください。」
        bot「（残留リマインダーの内容を一覧表示）」
    ３．[通常]の項番（２．）より処理を開始

    [削除]
    １．「@bot名[半角スペース]del」を入力」
    ２－A．リマインダーが設定されている場合
        bot「どのリマインダーを削除しますか？（番号で指定）」
        bot「複数削除する場合、半角スペースで区切って指定」
        bot「（設定リマインダーを一覧表示）」
        →番号で入力
        bot「（リマインダーの内容）が削除されました。」
    ２－B．リマインダーが1つも設定されていない場合
        bot「現在リマインダーは設定されていません。」

    [更新]
    １．「@bot名[半角スペース]upd」を入力」
    ２－A.リマインダーが設定されている場合
        bot「どのリマインダーを更新しますか？（番号を指定）」
        bot「※複数指定不可」
        bot「（設定リマインダーを一覧表示）」
        →番号で入力
        bot「
        bot「何時間後にリマインダーを設定しますか？」
        bot「例 : 3時間5分後の場合　→　03:05」
        bot「リマインドの種類を設定してください（数字で入力）」
        bot「リマインダーの内容を設定してください」
        →文字列入力
        bot「リマインダーの更新が完了しました。」
    ２－B.リマインダーが設定されていない場合
        bot「現在リマインダーは設定されていません。」

〇仕様部分
    ■リマインダー格納方法
        クラスを利用し、設定されるたびインスタンスを生成する
        →プロパティ
            ・discordID(登録者)
            ・リマインド時間(HH:MM)
            ・種類
            ・メッセージの内容
            ・投稿日時(YYYY/MM/DD HH:MM)
            ・リマインドされる時間(YYYY/MM/DD HH:MM)
                →投稿日時 + リマインド時間

    ■バリデーション対象
        １．コマンド
        ２．各入力内容

    ■jsonへの書き込みについて
        ・更新されるタイミング
            ・アプリが終了されるタイミング
            
        ・リマインダー.jsonの内容
            ・キー = 連番。
            ・値 = リマインダーの設定内容
                1つ目 : 登録者のdiscordID
                2つ目 : 時間(設定時に入力された時間)
                3つ目 : 種類
                4つ目 : 内容
                5つ目 : 設定した時点の日時
            ※区切り文字 = 「&segm&」
            　→「&segm&」をsplitで区切る

        -----サンプル.json-----
        {
            last_date : "yyyy/MM/dd HH:mm",

            reminder : {
                "1":"AAAAAA&segm&04:00&segm&交配&segm&TEKティラノ&segm&yyyy/MM/dd HH:mm",
                "2":"AAAAAA&segm&01:37&segm&孵化&segm&TEKティラノyyyy/MM/dd HH:mm",
                "3":"BBBBBB&segm&06:00&segm&その他&segm&ギガノト偵察&segm&yyyy/MM/dd HH:mm"
            }
        }

        ※last_date → BOTが終了する直前の日時を記録
        ※reminder → リマインダーの内容
